# websocket-dom

Experimental deterministic sync between backend JSDOM and frontend DOM using WebSockets.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.26. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
